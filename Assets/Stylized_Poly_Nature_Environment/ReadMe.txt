Thank you for your support and for choosing the Stylized Poly Nature Environment!

The asset pack is split into two [.unitypackage] files, one for each Render Pipeline.

-If you're using [Built-In 3D] render pipeline, you need to DOUBLE-CLICK on the [stylized_poly_nature_environment_built-in.unitypackage] and import it.

-If you're using [Universal 3D/ URP] render pipeline, you need to DOUBLE-CLICK on the [stylized_poly_nature_environment_urp.unitypackage] and import it.

After importing, follow the [Online Documentation] to learn how to use the assets properly:
https://lmhpoly.notion.site/Stylized-Poly-Nature-Environment-2f9afeb841fd807f8443f21f65db703d
